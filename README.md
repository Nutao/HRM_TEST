HRM_TEST
-----------
一个简单的人员管理系统，由java和mysql实现

主要功能：

> 1. 对员工信息的增加
> 2. 对员工信息的修改
> 3. 对员工信息的删除
> 4. 查看数据库中所有员工的信息
> 5. 通过关键字检索员工的信息记录(通过与所有的字段进行匹配进行检索)！
